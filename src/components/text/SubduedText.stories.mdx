# SubduedText

Use `<SubduedText/>` to render subdued text for the Hibana version of the app. In the OG theme, the
text renders as a default paragraph.

## Props

<Props of={SubduedText} />

## Examples

<Preview>
  <SubduedText>Hello, world</SubduedText>
</Preview>

## Usage

```js
import React from 'react';
import { SubduedText } from 'src/components/text';

const MyComponent = () => <SubduedText>Hello, world</SubduedText>;
```
