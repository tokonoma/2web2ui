# ToggleBlock

Use the `<ToggleBlock/>` to group a visible label with the Matchbox `<Toggle/>` component in to a
row.

## Examples

<Preview>
  <Panel>
    <Panel.Section>
      <ToggleBlock
        input={{ name: 'my-toggle-block' }}
        label="My Toggle Block"
        helpText="This is some help text"
      />
    </Panel.Section>
  </Panel>
</Preview>

## Usage

```js
import React from 'react';
import ToggleBlock from 'src/components/toggleBlock/ToggleBlock';

const MyComponent = () => (
  <ToggleBlock
    input={{ name: 'my-toggle-block' }}
    label="My Toggle Block"
    helpText="This is some help text"
  />
);
```
