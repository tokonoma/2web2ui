# Empty

Use the `<Empty/>` component for empty `<Panel/>` component states.

## Props

<Props of={Empty} />

## Examples

### Default

<Preview>
  <Story name="default">
    <Panel>

    </Panel>
  </Story>
</Preview>

## Usage

```js
import React from 'react';
import Empty from 'src/components/empty/Empty';

const MyComponent = () => (
  <Panel>
    <Empty message="No IP pools found." />
  </Panel>
);
```
